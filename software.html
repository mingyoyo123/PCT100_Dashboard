1. Wi-Fi 配网指南

PCT-100 提供了基于串口（Baud: 115200）的交互式配置菜单，无需修改代码重新编译即可更换网络环境。

========== 设备交互菜单 ==========
  1. 扫描并连接 WiFi (自动保存)
  2. 清除已保存的 WiFi 网络
  3. MQTT 远程服务器高级配置指南
==================================
请选择 (1/2/3):

输入 1 后，系统将扫描并列出带编号的网络列表，输入 c <编号> <密码> 即可连接，凭证会自动写入 ESP32 NVS 闪存中。

2. MQTT 高级设定

系统运行中支持动态修改物联网接入点：

  • mqtt_info：打印当前生效的 IP、端口、账号和设备 ID。
  • mqtt_ip 47.98.170.180：修改 MQTT 服务器地址。
  • mqtt_id PCT_100_005：更改本设备的唯一标识符。
  • mqtt_apply：保存以上所有修改并重启设备。

3. Profile (配置方案) 快速切换

为了方便在不同网络环境之间频繁切换，固件加入了配置预设功能：

保存方案： mqtt_save_profile config1
加载方案： mqtt_load_profile config2
查看所有方案： mqtt_list_profiles
