MQTT 远端配置与 JSON 协议

PCT_100 支持通过串口免刷机动态更改通信核心参数，所有凭证将永久保存在 NVS (Non-Volatile Storage) 闪存中。

1. 串口配置指令 (Baud: 115200)

  • mqtt_ip <服务器IP> - 修改 MQTT Broker 地址
  • mqtt_port <端口> - 修改通信端口 (默认: 8081)
  • mqtt_user <账号> - 修改平台认证用户名
  • mqtt_apply - 强制保存并重启连接

2. 标准 JSON 下行控制协议

上位机请向 chemctrl/<设备ID>/command 主题发布以下 JSON 格式载荷以远程接管设备：

A. 远程控制继电器 (仅限手动模式)

{
  "cmd": "set_relay",
  "relay": 3,
  "value": true
}

B. 动态修改环境防灾阈值

{
  "cmd": "set_threshold",
  "temp": 35.5,
  "light": 160.0
}
